# benrydal.com

Personal portfolio website for Ben Rydal Shapiro, showcasing research, projects, software, and publications.

**Live site:** [benrydal.com](https://benrydal.com)

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/) with Svelte 5
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
# Install dependencies
yarn install

# Start dev server
yarn dev

# Production build
yarn build

# Preview production build
yarn preview
```
